# NaturalVelocity

Modul integrasi Proxy (Velocity) untuk server **NaturalSMP**. Menangani konektivitas, bridging pesan, dan sinkronisasi data global.

## ✨ Fitur Utama
*   **Advanced Proxy**: Optimasi routing pemain antar-server.
*   **Global Messaging**: Jembatan komunikasi antar-server (Bukkit <-> Velocity).
*   **Unified Logic**: Menangani permission dan data global di level proxy.
